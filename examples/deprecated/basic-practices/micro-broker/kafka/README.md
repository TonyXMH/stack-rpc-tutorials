# Kafka Pubsub

本篇演示如何使用Kafka消息中间件来编写Micro的Pubsub服务，本篇由[Jayn Yang](https://github.com/jayn1985)提供，略作结构上的删改。

## 预置条件：

安装 Kafka

## 目录

- [cli](cli) 客户端
- [srv](srv) 服务端

## 运行程序

### 客户端

```bash
cd cli
go run client.go
```

### 服务端

```bash
cd srv
go run server.go
```
