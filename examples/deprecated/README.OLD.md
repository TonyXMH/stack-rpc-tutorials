# micro 最佳实践（已废弃）

【重要】本教程因Go-Micro已经毕业，不再更新，谢谢大家支持。请稳步社区主导基于Go-Micro开发的框架: [Stack-RPC](https://github.com/stack-labs/stack-rpc)【重要】

本项目旨在以全面的视角向大家演示micro服务的开发，并尽可能介绍其功能。我们尝试由浅入深，从基础篇开始，每个目录对应一个级别的示例。
